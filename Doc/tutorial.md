# ScholarSlide AI — 코드 점검 및 기능 요약

## 코드 점검 결과 및 수정 사항

### 1. 수정한 버그 (index.js)

1. **ReferenceStore 미구현**  
   - `ReferenceStore` IIFE 안에 메서드(`add`, `remove`, `getAll`, `clear`)와 `return`이 없어 참고문헌 추가/삭제가 동작하지 않을 수 있었습니다.  
   - **조치:** `js/store/references.js`와 동일한 구현을 `index.js`에 넣었습니다.

2. **인용 포맷터·저장 목록 미정의**  
   - `formatCitation`, `formatInText`, `getSavedRefList`, `saveRefList`, `addToSavedList`가 `index.js`에 없고, HTML은 `index.js`만 불러옵니다.  
   - **조치:** APA/Chicago/MLA 포맷과 저장 목록 함수를 `index.js`에 추가했습니다.

3. **parseAPA / addParsedRef 누락**  
   - 참고문헌 모달의 "APA 파싱" 탭에서 `parseAPA()`와 `addParsedRef()`를 사용하는데, `index.js`에 정의가 없었습니다.  
   - **조치:** `parseAPA()`와 `addParsedRef()`를 `index.js`에 구현해 두었습니다.

4. **전역 상수·상태 변수 누락**  
   - `LS_ACTIVE_KEY`, `LS_KEYS_LIST`, `LS_SESSIONS`, `LS_SAVED_REF_LIST`와 `rawText`, `slides`, `slideStyle` 등이 `index.js`에 선언되어 있지 않았습니다. (다른 js 파일에만 존재)  
   - **조치:** `index.js` 상단 STATE 블록에 위 상수와 전역 상태 변수 선언을 추가했습니다.

이제 **index.html에서 index.js만 로드해도** 참고문헌·세션·자동저장·API 키 저장이 정상 동작하도록 맞춰 두었습니다.

## ScholarSlide AI v4.35 — 기능 요약

| 구분 | 기능 |
|------|------|
| **진입** | PDF/DOCX/TXT 업로드 또는 텍스트 붙여넣기 → 원문 추출 |
| **요약** | **전체 요약**: 원문 기반 구조화 요약(배경, 방법론, 발견, 시사점 등). **슬라이드 생성**: 슬라이드 수·표지 포함·문체(학술체/존댓말) 설정 후 JSON 슬라이드 배열 생성 |
| **번역** | 요약/원문을 한국어로 번역(캐시 유지), 번역 결과 팝업 보기·복사 |
| **슬라이드** | 제목·불릿·노트·표지 플래그·`visPrompt`(이미지용) 편집, 빈 슬라이드 추가, **AI 슬라이드 1장 추가**(문서/키워드 기반), AI 재작성·확장, 삭제, 썸네일 선택. **실행 취소/다시 실행**(Ctrl+Z / Ctrl+Shift+Z): 이동·사이즈 조절·이미지·텍스트 편집 등 |
| **MD 에디터** | 마크다운 편집(H1~H3, 굵게/기울임/취소선, 목록, 코드, 링크, YouTube), 색·글자 크기, 미리보기, "로드/적용"으로 슬라이드와 동기화 |
| **이미지** | 업로드 이미지 크롭(비율: 자유/전체/1:1/4:3/16:9 등), **AI 이미지 생성**(Gemini, `visPrompt`), 일괄 시각화(백그라운드 진행률), 갤러리에서 다운로드, 슬라이드당 이미지 2장·AI 수정 |
| **참고문헌** | 직접 입력, **APA 문장 파싱**, **Scholar AI 검색**(Gemini로 논문 5편 JSON), 저장 목록(로컬)·필터·삭제, APA/Chicago/MLA 인용, 본문 인용 삽입, 참고문헌 슬라이드/전체 슬라이드 추가, JSON 내보내기·가져오기 |
| **발표** | **발표 모드**: 풀스크린, 이전/다음, 줌·맞춤, 발표자 노트 표시. **펜 도구**: 포인터/펜/형광펜/지우개, 색·굵기·농도. **외부 창 발표**: 새 창에서 슬라이드+원고 재생, **폰트 크기 조절**(A−/A+ 버튼, Ctrl+Alt+−/+) |
| **저장/불러오기** | **자동저장**: 30초마다 IndexedDB. **불러오기**: 자동저장 복구 / 저장된 세션(localStorage) / **프로젝트 파일(.ssp)**. **프로젝트**: 이름 입력 후 "파일로 내보내기(.ssp)" 또는 "세션으로 저장" |
| **내보내기** | **PPTX**: PptxGenJS, 발표 원고·이미지·참고문헌 슬라이드 옵션, 한/영 폰트 선택 |
| **기타** | **Gemini API 키**: 설정·저장·여러 키 관리. **확인 모달**: 요약/슬라이드/원고/이미지 생성 전 확인. **PDF 미리보기**: 업로드 후 썸네일·페이지 이동·줌. **전역 진행률**: 헤더 좌측 바. **초기화**: 전체 데이터·자동저장 삭제 |

## 프로젝트 구조 참고

- **index.html** + **index.js** 한 세트로 동작하도록 수정했으며, `index.html`은 `index.js`만 로드합니다.
- `js/` 폴더의 `state/state.js`, `store/references.js`, `utils/citations.js` 등은 **현재 HTML에서 로드되지 않습니다.**  
  빌드/번들로 합치는 구조가 아니라면, 지금은 **index.js에 필요한 내용을 반영한 상태**가 기준입니다.
