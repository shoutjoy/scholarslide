# 뷰어 새창 기능 점검 체크리스트

새창(뷰어)이 열린 뒤 모든 버튼이 정상 동작하는지 확인하는 체크리스트입니다.

## 수정 요약 (2025-03)

- **원인**: `document.write`로 주입되는 인라인 스크립트가 `contentHtml` 등 사용자 콘텐츠 때문에 파싱 오류로 실행되지 않음
- **해결**: 뷰어 전용 로직을 `js/ui/viewer-standalone.js`로 분리, 인라인 스크립트 최소화
- **추가**: `<base href>`로 about:blank 창에서 상대 경로 스크립트가 올바르게 로드되도록 설정

## 문제 발생 시 확인 사항

1. **브라우저 콘솔(F12)** 에 스크립트 로드 오류가 있는지 확인
2. **viewer-standalone.js** 가 로드되는지 Network 탭에서 확인
3. `index.html` 을 **로컬 파일(file://)** 로 열었다면, 일부 브라우저에서 `js/ui/viewer-standalone.js` 상대 경로가 제대로 동작하지 않을 수 있음 → 로컬 서버로 실행 권장
