# Q&A — 추가된 이미지의 배경 제거

## 질문

**추가된 이미지의 배경 제거가 가능한가?**

---

## 답변

**아니요. 현재 ScholarSlide에서는 추가된 이미지의 배경을 제거하는 기능이 없습니다.**

### 현재 이미지 관련 기능

- **이미지 업로드 & 크롭**  
  JPG, PNG, GIF, WebP 업로드 후, 비율(자유, 1:1, 4:3, 16:9 등)로 **크롭**만 가능합니다. 이미지 모달에는 “크롭 적용”, “AI 이미지 재생성”만 있고 “배경 제거” 버튼/메뉴는 없습니다.

- **AI 이미지 재생성**  
  `aiEditImage()`로 Gemini를 이용해 이미지를 다시 생성하는 기능만 있으며, 배경 제거(세그멘테이션/투명 처리)와는 별개입니다.

- **배경 제거 관련 코드**  
  코드베이스에는 “배경 제거”, `background removal`, `removebg`, `remove.bg`, 이미지 배경 제거용 API 호출 등이 없습니다.

### 정리

- **추가된 이미지의 배경 제거는 현재 지원되지 않습니다.**  
- 구현하려면 remove.bg 같은 **배경 제거 API** 연동, 또는 브라우저에서 동작하는 **이미지 세그멘테이션/마스킹** 로직을 새로 추가해야 합니다.
