# ScholarSlide AI

논문·원고를 업로드하면 **AI가 요약하고 슬라이드를 만들어 주는** 학술 발표 자료 제작 앱입니다.  
Gemini API로 요약, 슬라이드 생성, 발표 원고, 참고문헌 검색, AI 이미지 생성까지 한 곳에서 처리할 수 있습니다.

---

## ⚠️ API 키 필요 (필수)

**본 앱의 AI 기능(요약, 슬라이드 생성, 이미지 생성, 참고문헌 검색 등)을 사용하려면 Google AI Studio에서 발급한 API 키가 필요합니다.**

1. **[Google AI Studio](https://aistudio.google.com)** 에 접속합니다.
2. Google 계정으로 로그인한 뒤 **Get API key** 또는 **API 키 만들기**를 선택합니다.
3. 발급된 API 키를 복사합니다. (형식: `AIza...` 로 시작)
4. 앱 실행 후 **헤더의 🔑 API 키 설정** 버튼을 누르고, 복사한 키를 입력·적용합니다.

> 💡 API 키는 브라우저 로컬에만 저장되며, 서버로 전송되지 않습니다.  
> 이미지 생성 시에는 동일한 키를 사용하거나, AI Studio에서 이미지 생성용 키를 별도로 설정할 수 있습니다.

---

## 주요 기능

### 📄 문서 → 슬라이드
- **PDF / DOCX / TXT** 업로드 또는 텍스트 직접 붙여넣기
- **전체 요약**: 배경, 방법론, 발견, 시사점 등 구조화 요약
- **슬라이드 생성**: 슬라이드 수·표지 포함·문체(학술체/존댓말) 설정 후 AI가 JSON 슬라이드 배열 생성
- **한국어 번역**: 요약/원문 번역 후 보기·복사

### ✏️ 슬라이드 편집
- 제목·불릿·노트·표지 플래그·이미지용 설명(visPrompt) 편집
- 빈 슬라이드 추가, **AI 슬라이드 1장 추가**(문서/키워드 기반), AI 재작성·확장
- **실행 취소/다시 실행** (Ctrl+Z / Ctrl+Shift+Z): 이동·사이즈 조절·이미지·텍스트 편집
- 마크다운 에디터: H1~H3, 굵게/기울임, 목록, 코드, 링크, YouTube, 슬라이드와 동기화

### 🖼️ 이미지
- 업로드 이미지 **크롭** (비율: 자유/전체/1:1/4:3/16:9 등)
- **AI 이미지 생성** (Gemini, visPrompt), 일괄 시각화, 갤러리 다운로드
- 슬라이드당 이미지 2장, AI로 이미지 수정

### 📚 참고문헌
- 직접 입력, **APA 문장 파싱**, **Scholar AI 검색**(논문 5편 JSON)
- 저장 목록·필터·APA/Chicago/MLA 인용, **본문 인용 삽입**
- 참고문헌 슬라이드·전체 슬라이드 추가, JSON 내보내기·가져오기

### 🎤 발표
- **발표 모드**: 풀스크린, 이전/다음, 줌·맞춤, 발표자 노트, 폰트 크기 조절
- **펜 도구**: 포인터/펜/형광펜/지우개, 색·굵기·농도
- **외부 창 발표**: 새 창에서 슬라이드+원고 재생, 폰트 A−/A+ 조절

### 💾 저장·내보내기
- **자동저장**: 30초마다 IndexedDB
- **불러오기**: 자동저장 복구 / 저장된 세션 / **프로젝트 파일(.ssp)**
- **PPTX 내보내기**: 발표 원고·이미지·참고문헌 슬라이드 옵션, 한/영 폰트 선택

## 시작하기

1. **API 키 발급**: [Google AI Studio](https://aistudio.google.com)에서 API 키를 발급받습니다.
2. **앱 실행**: `index.html`을 브라우저에서 열거나, 로컬 서버로 실행합니다.
3. **API 키 설정**: 헤더의 🔑 **API 키 설정**에서 키를 입력·적용합니다.
4. **문서 업로드**: PDF/DOCX/TXT를 업로드하거나 텍스트를 붙여넣은 뒤 **요약** → **슬라이드 생성**을 진행합니다.

---

## 단축키

- **헤더 ❓ Help** 버튼을 누르면 전체 단축키 목록을 볼 수 있습니다.
- 슬라이드: `Ctrl+Z` 실행 취소, `Ctrl+Shift+Z` 다시 실행
- 발표: `F5` 처음부터, `Shift+F5` 현재부터, `←`/`→`/`Space` 이동, `Esc` 종료

---

## 기술 구성

- **프론트**: HTML5, CSS3, JavaScript (Vanilla)
- **AI**: Google Gemini API (요약, 슬라이드, 원고, 참고문헌 검색, 이미지 생성)
- **문서 처리**: PDF.js, Mammoth.js (PDF/DOCX 텍스트 추출)
- **저장**: IndexedDB(자동저장), localStorage(세션·API 키·참고문헌)
